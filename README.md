# Project Title Spell Crafting System
This project is a typescript spell casting system that allows players to construct spells using casting types, elements, and behaviors, and uses a stat creation system to show users the stats of their spells.

## Instructions for Build and Use

Steps to build and/or run the software:

1.	Install the correct version of Node.js and TypeScript.
2.	Import the project into an IDE such as Visual Studio Code or IntelliJ.
3.	Compile the TypeScript files using the command tsc.
4.	Run the project using a local web server or your IDE’s built in preview.
5.	Open the HTML interface to interact with the spellcrafting UI.

Instructions for using the software:
1.	Launch the project in a browser.
2.	Select a casting type, an element, and one or more behaviors.
3.	Click “Create Spell” to view the combined stats and description.
## Development Environment
To recreate the development environment, you need the following software and libraries:
1.	Node.js
2.	TypeScript
3.	Visual Studio, or another IDE that can run TypeScript
4.	A simple local web server (live server, Vite, etc.)
